## 🚀 Demo Mode Explained

### What is Demo Mode?
Frontend works independently without backend API:
- ✅ Mock data for hospitals, doctors, hotels
- ✅ Demo login/register (any credentials work)
- ✅ Demo form submission (shows success alert)
- ✅ All UI fully functional

### Why?
Allows frontend development while backend is being built.

### When Backend Ready?
- Update `.env` with `VITE_API_BASE_URL`
- Demo mode automatically replaced with real API calls
- Zero code changes needed
